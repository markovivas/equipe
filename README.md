# Equipe de Trabalho

Plugin WordPress para exibir os membros da equipe com nome, secretaria, setor, profissão, função, foto e contato.

## Descrição

Gerencie os membros da equipe diretamente pelo painel do WordPress. Os dados são salvos no banco de dados e exibidos via shortcode.

- Gerenciamento completo via **Admin do WordPress** (menu "Equipe")
- Upload de fotos pela **Biblioteca de Mídia**
- Busca por nome e filtros por secretaria, setor e função
- Filtro por letra do alfabeto
- Paginação automática
- Interface moderna e responsiva

## Instalação

1. Faça upload da pasta do plugin para `wp-content/plugins/`
2. Ative o plugin em **Plugins > Plugins instalados**
3. Acesse o menu **Equipe** no admin para cadastrar os membros
4. Adicione o shortcode `[membros_empresa]` na página desejada

## Uso

```
[membros_empresa]
```

## Gerenciamento

Vá em **Equipe** no menu do WordPress para:
- **Adicionar** novos membros com foto, dados de contato e localização
- **Editar** informações de membros existentes
- **Excluir** membros

## Funcionalidades

- Busca por nome
- Filtros por secretaria, setor e função
- Filtro alfabético (A-Z)
- Paginação automática (4 por página)
- Status online/offline
- Ícones de e-mail e telefone
- Design responsivo
